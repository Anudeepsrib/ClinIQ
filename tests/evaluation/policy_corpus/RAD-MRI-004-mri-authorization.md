# MRI Authorization And Coverage Reference

This synthetic policy is for ClinIQ evaluation only.

## Non-Urgent MRI

A non-urgent MRI request requires an ordering provider note, documented indication, payer authorization checklist, and confirmation that the imaging site is covered by the patient's plan.

The scheduling team must record payer authorization status before appointment confirmation. Missing payer authorization should route to the radiology authorization queue.

## Urgent Exception

An urgent MRI exception requires review by the radiologist on call and ordering provider attestation. The exception note must state why routine authorization timing would create operational or patient-safety risk.

This policy is an administrative authorization reference. It does not decide whether an MRI is clinically appropriate for a patient.
